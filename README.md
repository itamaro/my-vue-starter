# My Vue Starter

A small Vue.js + Vuex + VueMaterial starter project

To start:

```bash
$ yarn install
```

To develop:

```bash
$ yarn run dev```

To build for production:

```bash
$ yarn run build
```

To lint you code:

```bash
$ yarn run lint
```
